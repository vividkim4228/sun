## git 

1. 내컴퓨터 history (백업)
2. 웸상의 어딘가에 자료가 기록
3. 또다른 컴퓨터에 자료를 배포/동기화

---

- 무슨 데이터를 올릴것인가?
- 올린데이터가 어떤한 내용을 담고있는가?
- 웹에 전송해서 백업
- 다른 컴에서는 올라온자료를 받아서 사용?

---

`$ git clone 웹주소`  : 최초 복제

`$ git pull`  : 기존자료 동기화해서 가져오기 

`git push`  : 내 파일 올리기

`git status`  : 자료상태 보기

`git add 첨부자료(폴더, 파일)`  : 올릴자료 선택

`git commit -m "첨부자료 설명(영/한)"` : 첨부자료 설명

`git remove 뺄자료(폴더, 파일)`  : 뺄 자료 선택

`git reset ` : 초기화
