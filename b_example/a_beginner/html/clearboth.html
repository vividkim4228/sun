clear 기능을 사용하기위해 after의 개념을 익혀보자

before, after요소는 가상클래스라고 하며, 실제로 존재하지 않는 요소로서
선택자의 내부의 자식으로 가상으로 존재하도록 만들어서 처리하게 하는 기능
- 기본적으로 내용을 담는 content:""; 를 사용하고
- 원래 inline 요소이며, 필요시 강제로 해당내용을 display:block으로 block 처리하며 float 기능을 상쇄하는 처리를 한다
- 가상클래스는 실제로 내용을 읽어주지 않는다. 눈에는 보인다
.clearfix::after{content:" "; display: block; width: 0; height: 0; clear: both;} 와 같이 적용
